# Travel-Connect-

Welcome to **Travel Connect**, a premium, modern, and professional travel booking platform.

## Features
- **Modern UI/UX**: Built with premium styling, smooth transitions, and glassmorphism elements.
- **Multiple Flows**: Integrated customer booking flows, service provider dashboards, and admin panels.
- **Dynamic Themes**: Support for light and dark themes with fully readable fonts.
- **Vite & React**: Fast development server and builds.

## Technology Stack
- Core: React, TypeScript, Vanilla CSS
- Build Tool: Vite
